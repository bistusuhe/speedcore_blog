## 项目简介

Neon Drift 是我练习 Unity 手感打磨与 Shader 的作品。玩家在赛博朋克都市中跑酷，躲避障碍、收集能量。

## 我的职责

- 全部玩法与关卡程序化生成
- 自研后处理描边 + Bloom Shader
- 触屏 / 键盘双输入适配

## 技术亮点

### 程序化关卡

用 Wave Function Collapse 思想生成无尽轨道，保证可达性与节奏感。

### 自研 Shader

```hlsl
half4 frag(v2f i) : SV_Target {
    half4 col = tex2D(_MainTex, i.uv);
    half edge = 1 - saturate(abs(dot(normalize(i.viewDir, i.normal)));
    col.rgb += _NeonColor * pow(edge, _Power);
    return col;
}
```
