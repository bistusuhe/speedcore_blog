## 项目简介

校企合作项目，为企业提供拖拽式大屏搭建能力，内置 30+ 图表组件与多数据源接入。

## 技术栈

- 前端：Vue 3 + TypeScript + ECharts + Fabric.js
- 后端：Spring Boot + MyBatis Plus + Redis
- 数据源：MySQL / ClickHouse / REST API

## 我的职责

- 负责画布渲染引擎（基于 Fabric.js 二次封装）
- 设计组件物料系统与拖拽协议
- 接入实时数据推送（WebSocket）
